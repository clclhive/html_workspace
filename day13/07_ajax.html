AJAX (Asynchronous JavaScript and XML)

AJAX는 웹 페이지 전체를 새로고침하지 않고도, 백그라운드에서 서버와 데이터를 교환하여 페이지의 일부만 동적으로 업데이트하는 기술입니다.
이를 통해 사용자는 더 빠르고 부드러운 웹 경험을 할 수 있습니다.

동기(Synchronous) vs 비동기(Asynchronous)

동기 방식: 서버에 요청을 보내고 응답이 올 때까지 모든 작업이 멈춥니다. (예: 페이지 이동)
비동기 방식: 서버에 요청을 보낸 후, 응답을 기다리지 않고 다른 작업을 계속 수행합니다. 응답이 오면 그때 처리합니다. (AJAX의 핵심)
