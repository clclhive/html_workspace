함수의 파라미터(Parameter)와 인자(Argument)

함수를 정의할 때 사용하는 변수를 파라미터(Parameter, 매개변수)라고 합니다.
함수를 호출할 때 전달하는 실제 값을 인자(Argument, 인수)라고 합니다.
파라미터는 함수가 외부로부터 값을 받아 내부에서 사용할 수 있도록 하는 '통로' 역할을 합니다.

1. 기본 파라미터 사용법

가장 일반적인 형태로, 함수를 정의할 때 소괄호 안에 파라미터 이름을 나열합니다.

2. 기본값(Default) 파라미터

함수 호출 시 인자가 전달되지 않았을 경우를 대비해 파라미터에 기본값을 설정할 수 있습니다. (ES6+ 문법)

3. 나머지(Rest) 파라미터

정해지지 않은 개수의 인자들을 배열로 받고 싶을 때 사용합니다. 파라미터 이름 앞에 `...`를 붙입니다.
나머지 파라미터는 항상 마지막 파라미터여야 합니다.
