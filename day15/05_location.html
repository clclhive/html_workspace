`window.location` 객체 예제

window.location 객체는 현재 페이지의 URL 정보를 담고 있으며, 페이지를 새로고침하거나 다른 URL로 이동하는 등의 브라우저 탐색 기능을 제어할 수 있게 해줍니다.

1. `location.href` (페이지 이동 - 기록 남김)

location.href = "URL" 또는 단순히 location = "URL"은 지정된 URL로 페이지를 이동시킵니다. 이전 페이지는 브라우저의 방문 기록에 남아 '뒤로 가기' 버튼으로 돌아갈 수 있습니다.

2. `location.assign()` (페이지 이동 - 기록 남김)

location.assign("URL")은 location.href와 동일하게 지정된 URL로 페이지를 이동시키고, 이전 페이지를 방문 기록에 남깁니다. 기능적으로는 location.href와 거의 같습니다.

3. `location.replace()` (페이지 이동 - 기록 대체)

location.replace("URL")은 지정된 URL로 페이지를 이동시키지만, 현재 페이지를 브라우저의 방문 기록에서 새로운 페이지로 대체합니다. 따라서 '뒤로 가기' 버튼을 눌러도 이전 페이지로 돌아갈 수 없습니다. (로그인 후 리다이렉트 등에 유용)

4. `location.reload()` (페이지 새로고침)

location.reload()는 현재 페이지를 새로고침합니다. 인자로 true를 전달하면 서버로부터 페이지를 강제로 다시 로드하고, 생략하거나 false를 전달하면 브라우저 캐시를 활용할 수 있습니다.
